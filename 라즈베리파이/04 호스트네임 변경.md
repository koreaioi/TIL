# 호스트네임 변경

```bash
sudo su - #root로 로그인
raspi-config #라즈베리파이 하드웨어 설정으로 이동
```

- System Options 선택
- Hostname 선택

- sudo reboot (혹은 알아서 리부팅)
- 일정 시간 후 재 접속
